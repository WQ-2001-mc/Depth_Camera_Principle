# Depth Camera Principle｜深度相机原理资料库

这是一个面向学习、研发和硬件选型的深度相机中文资料库，围绕“物理原理—算法流程—误差来源—计算与带宽瓶颈—系统选型”组织内容。

## 阅读入口

- [深度相机原理、算法与系统瓶颈](深度相机原理介绍.md)：主文档，含完整 Mermaid 分类思维导图；
- [代表资料索引](references/资料索引.md)：8 篇代表性资料的元数据、来源、许可说明和校验值；
- [主动双目资料导航](docs/03_主动双目/README.md)：散斑/随机纹理与条纹/编码纹理的关系；
- [ToF 总览](docs/04_ToF/README.md)：dToF 与 iToF 的关系及子目录导航。

## 核心分类

本仓库重点覆盖五条物理测距路线：

1. 被动双目；
2. 单目结构光；
3. 主动双目；
4. dToF；
5. iToF。

单目深度学习、多目视觉、扫描 LiDAR、事件相机和多传感器融合在主文档的技术全景中说明，但不作为本轮代表 PDF 的独立分类。
